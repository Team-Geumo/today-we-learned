### 절차형 프로그래밍

- 절차형 프로그래밍
  
  - 로직이 수행되어야 할 연속적인 계산 과정으로 이루어져 있음
  
  - 예시
    
    ```js
    const ret = [1, 3, 5, 11, 33, 55, 22, 44, 35]
    let a = 0
    for (let i = 0; i < ret.length; i++) {
        a = Math.max(ret[i], a)
    }
    console.log(a) // 55
    ```
  
  - 장점
    
    - 코드의 가독성이 좋으며 실행 속도가 빠름
      
      - 계산이 많은 작업 등에 사용
  
  - 단점
    
    - 모듈화하기 어렵고 유지 보수성이 떨어짐
