### 자바의 상속과 구현

- 상속 (extends)
  
  - 자식 클래스가 부모 클래스의 메서드 등을 상속받아 사용하며 자식 클래스에서 추가 및 확장을 할 수 있음
  
  - 재사용성, 중복성의 최소화가 이루어짐

- 구현 (implements)
  
  - 부모 interface를 자식 클래스에서 재정의하여 구현
  
  - 상속과 달리 `반드시 부모 클래스의 메서드를 재정의`하여 구현해야 함

- 상속과 구현의 차이
  
  - 상속
    
    - 일반 클래스, abstract 클래스를 기반
  
  - 구현
    
    - interface를 기반으로 구현
